# Output Cleanup Preference

When the user asks to generate, download, convert, export, extract, build, or organize any files or folders, keep only the explicitly requested final output at the end of the task. Clean up helper artifacts created during the task, including temporary ZIP files, archives, extraction directories, cache files, draft files, preview files, log files, and intermediate build outputs. Before cleanup, confirm the cleanup will not delete files that already existed before the task or useful work files the user may still need.
